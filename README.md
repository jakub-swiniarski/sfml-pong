## How to install?
### Prerequisites
- g++
- make 
- sfml
### Compile & install
Run this with root privileges:
```shell
make install 
```
### How to run?
Run this command: 
```shell
pong
```
### How to uninstall?
Go inside the cloned repository and run this with root privileges: <br/>
```shell
make uninstall
```

## Controls
Up arrow - move up <br/>
Down arrow - move down <br/>
Escape - exit the game

## Customization
Edit src/config.hpp and run this with root privileges:
```shell
make install 
```
