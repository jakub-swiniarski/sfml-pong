## How to install?
### Prerequisites
- g++
- sfml
- git
- make
### Compile & install
```shell
git clone https://github.com/jakub-swiniarski/sfml-pong.git
cd sfml-pong
sudo make install 
```
### How to run?
Run this command: 
```shell
pong
```
### How to uninstall?
Go inside the cloned repository and run: <br/>
```shell
sudo make clean uninstall
```

## Controls
Up arrow - move up <br/>
Down arrow - move down <br/>
Escape - exit the game

## Customization
Edit src/config.hpp and run:
```shell
sudo make install 
```
